# e-commerce-backend

## Get the project up and running
```
mysql -u root -p

Enter PW when promted

source db/schema.sql

quit

npm run seed

npm start
```
### A mysql database/application backend for e-commerce site. App is built using Express, MySQL2, dotenv, and sequelize.
